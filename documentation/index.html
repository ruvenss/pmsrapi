📚 Overview

About PMSRAPI

PMSRAPI is a PHP-based micro-service REST API framework designed for building scalable microservices with minimal configuration. It provides a simple yet powerful approach to creating REST APIs with built-in database operations, webhooks, file handling, and event management.

🔐 Authentication

Bearer Token Authentication

All API requests require authentication using a Bearer token in the Authorization header.

Authorization: Bearer YOUR_TOKEN_HERE
ℹ️ Note: The token is configured during installation and stored in the parent directory configuration file.

Request Format

All requests must use Content-Type: application/json and include a JSON body with the following structure:

{
    "function": "endpoint_name",
    "parameters": {
        // endpoint-specific parameters
    },
    "payload": {
        // optional additional data
    }
}

Response Format

All successful responses return a JSON object with the following structure:

{
    "success": true,
    "data": {
        "values": { ... },
        "table_last_update": "2025-11-22 10:30:00"
    }
}

📡 API Endpoints

GET Read Operations

GET info
Get microservice information, version, status, and configuration details
GET select_rows
Select multiple rows from a database table with filtering and ordering
GET select_row
Select a single row from a database table
GET count_rows
Count rows in a table matching specified conditions
GET sum_rows
Sum values of a specific field with WHERE conditions
GET select_plot
Select data formatted for plotting/charting (X,Y coordinates)
GET tables_info
Get information about all tables in the database
GET downloadfile
Download a file from the server storage
GET webhooks
List all configured webhooks for a specific method

POST Create Operations

POST insert
Insert a new row into a database table
POST uploadfile
Upload a file to server storage using base64 encoding
POST webhook_add
Register a new webhook endpoint
POST universe_insert_node
Insert data into another microservice node in the universe

PUT Update Operations

PUT update
Update records in a database table
PUT webhook_update
Update an existing webhook configuration

DELETE Delete Operations

DELETE delete
Delete records from a database table
DELETE universe_delete_node
Delete data from another microservice node in the universe

💡 Request Examples

Get Microservice Info

curl --location --request GET 'http://localhost:8000/v1/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data '{
    "function": "info"
}'

Select Multiple Rows

curl --location --request GET 'http://localhost:8000/v1/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data '{
    "function": "select_rows",
    "parameters": {
        "table": "users",
        "fields": "*",
        "where": "status = 1",
        "orderby": "created_at DESC",
        "limit": "10",
        "groupby": ""
    }
}'

Insert New Record

curl --location --request POST 'http://localhost:8000/v1/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data '{
    "function": "insert",
    "parameters": {
        "table": "users",
        "values": {
            "name": "John Doe",
            "email": "john@example.com",
            "status": "1",
            "created_at": "NOW()"
        }
    }
}'

Update Records

curl --location --request PUT 'http://localhost:8000/v1/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data '{
    "function": "update",
    "parameters": {
        "table": "users",
        "values": {
            "name": "Jane Doe",
            "updated_at": "NOW()"
        },
        "where": "id = 123"
    }
}'

Delete Records

curl --location --request DELETE 'http://localhost:8000/v1/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data '{
    "function": "delete",
    "parameters": {
        "table": "users",
        "where": "id = 123"
    }
}'

Upload File (Base64)

curl --location --request POST 'http://localhost:8000/v1/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data '{
    "function": "uploadfile",
    "parameters": {
        "file_name": "document.pdf",
        "local_path": "/path/to/uploads"
    },
    "payload": {
        "base64": "BASE64_ENCODED_FILE_CONTENT"
    }
}'

Count Rows

curl --location --request GET 'http://localhost:8000/v1/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data '{
    "function": "count_rows",
    "parameters": {
        "table": "users",
        "where": "status = 1"
    }
}'

🗄️ Database Functions

Available Database Operations

Function Description Parameters
select_rows Select multiple rows with filtering table, fields, where, orderby, limit, groupby
select_row Select a single row table, fields, where, orderby
count_rows Count matching rows table, where
sum_rows Sum field values table, field, where
insert Insert new record table, values (associative array)
update Update existing records table, values (associative array), where
delete Delete records table, where
⚠️ Important: Always use proper WHERE clauses to prevent accidental mass updates or deletions. The framework validates all requests and sanitizes inputs to prevent SQL injection.

🔗 Webhooks

Webhook Management

PMSRAPI includes built-in webhook support for event-driven architectures. Webhooks can be registered, updated, and triggered on specific database events.

Add Webhook

curl --location --request POST 'http://localhost:8000/v1/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data '{
    "function": "webhook_add",
    "parameters": {
        "url": "https://example.com/webhook",
        "method": "POST",
        "header_key": "X-Custom-Header",
        "user": "webhook_user",
        "pass": "webhook_password",
        "events": ["insert", "update", "delete"]
    }
}'

List Webhooks

curl --location --request GET 'http://localhost:8000/v1/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data '{
    "function": "webhooks",
    "parameters": {
        "method": "POST"
    }
}'

⚙️ Installation & Setup

Requirements

  • Ubuntu Linux 24.10 or greater
  • PHP 8.3 or greater
  • PHP Extensions: mysql, mbstring, intl, xml, zip, sqlite
  • Bash
  • unzip utility
  • Root access

Quick Install

apt update
apt upgrade -y
apt install unzip
mkdir /home/my_microservice
cd /home/my_microservice
wget -q https://github.com/ruvenss/pmsrapi/archive/refs/tags/0.0.23.zip -O "pmsrapi.zip"
unzip -qq pmsrapi.zip && rm pmsrapi.zip
mv pmsrapi-0.0.23/* /home/my_microservice
./install.sh
✅ Success: Follow the setup wizard to configure your microservice, database connection, and authentication tokens.
